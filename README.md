# joy-ui
Some React components which have been designed for serving basic CRUD actions to manage data.

## Components
* ManageEntity
* DataTable
* Modal/ModelProvider
* Spiner
* Inputs (Use with yup & formik)
    * TextBox
    * TextMulti
    * DatePicker

## Install
```shell
npm joy-ui
```

## Use
See in example

## Run demo
```shell
$ git clone https://github.com/TamVoMinh/joy-ui.git
$ cd joy-ui
$ npm install
$ yarn start:api
Open new terminal with same directory
$ yarn start
```
## Build component
```shell
yarn build:component
```
